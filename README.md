# Mini_project
🕹. 가위바위보 게임 만들기
1. 기능 
+ ✌, ✊, 🖐 순으로 화면에 출력
+ 버튼을 클릭하면 화면이 멈추며 이김.짐.비김의 유무가 alert창으로 출력
+ 다시하기 버튼을 누르면 게임을 다시할 수 있는 기회가 주어짐
+ 이김의 유무에 따라 점수가 1점씩 올라감.
